## Teste

**A hosted REST-API ready to respond** projeto de teste. 
https://reqres.in/
<br/>

#### Requirements

- gradle >= 4.7

```
gradle wrapper --gradle-version 4.7
```

#### Getting Started

Install the dependencies, go to the directory `./gatling` and execute;
```
gradle build
```

Run tests;

```
./gradlew gatlingRun
```
