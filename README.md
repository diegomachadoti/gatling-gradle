## QA NINJA

**Qa ninja** projeto de teste. <br/>

#### Requirements

- gradle >= 4.7

```
gradle wrapper --gradle-version 4.7
```

#### Getting Started

Install the dependencies, go to the directory `./gatling` and execute;
```
gradle build
```

Run tests;

```
./gradlew gatlingRun
```

#### Report
The reports are available in `gatling/build/reports/gatling`
